# 07 — BASE DE REFERÊNCIAS

Este documento **não gera conteúdo**. Ele define de onde o conhecimento pode vir e em que ordem. A internet nunca é a fonte primária: ela serve para *evoluir a Base*, sob aprovação — nunca para escrever um post direto.

## Hierarquia de conhecimento (ordem obrigatória)

1. **Base de Conhecimento (doc 06)** — fonte principal. Conceito, limitações, analogias, afirmações permitidas/proibidas, casos, objeções, ganchos.
2. **Manifesto (01) + Norte (00)** — o conteúdo reforça o posicionamento? Se não, não continua.
3. **Linguagem e Comunicação (03)** — traduz para a língua da marca.
4. **Frameworks (04A → 04B)** — escolhe objetivo e estrutura.
5. **Conhecimento interno do modelo** — só para melhorar exemplos, explicar conceitos estáveis e organizar o texto. **Nunca** para número, percentual, case ou comportamento atual de plataforma (Regra de Evidência do doc 05). Nunca substitui a Base.

## Conhecimento permanente vs. temporário

- **Permanente** (muda pouco → vive na Base): atribuição, UTM, pixel, ROAS, server-side, not set, GTM, mensuração, deduplicação, modelos de atribuição.
- **Temporário** (muda → não entra direto em conteúdo): atualização da Meta/GA4/Google, nova política, mudança no Consent Mode, novo recurso do TikTok. Antes de virar conteúdo, passa por Artur e, se fizer sentido, entra na Base como verbete/atualização.

## Regra da internet

Não consultar automaticamente. Só quando Artur pedir explicitamente ("pesquise as mudanças recentes do GA4", "verifique a documentação da Meta", "atualize a Base sobre Consent Mode"). Nesses casos a pesquisa serve para **atualizar a Base**, não para escrever o post.

## Fluxo de lacuna (gap)

Se, ao consultar a Base, o verbete não cobrir um ponto importante, o Claude **não inventa**. Ele avisa: *"Este verbete não cobre X. Deseja que eu pesquise em fontes oficiais para atualizar a Base antes de criar o conteúdo?"* Após aprovação → consulta fontes autorizadas → propõe atualização da Base → só então gera o conteúdo.

## Fontes autorizadas

**Primárias:** documentação oficial de Google Analytics, Google Tag Manager, Meta, Google Ads, Pinterest, TikTok, Consent Mode.

**Secundárias:** casos próprios, auditorias próprias, testes internos, artigos técnicos validados.

**Proibidas como autoridade:** blogs sem documentação, threads virais, vídeos sem referência, estatísticas sem origem, "especialistas" sem fonte verificável.
